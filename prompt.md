  assign tasks to each agent, and produce architecture, UI, and documentation deliverables

  Healthcare Credentialing System Build
  Goal:
  Design and implement an enterprise‑grade Healthcare Provider Credentialing System that automates the end‑to‑end process from provider application through
   network onboarding.

  Workflow:

  Provider Applies

  Retrieve CAQH Profile

  Verify Licenses (State Board)

  Check Sanctions/Exclusions (OIG, SAM, State)

  Verify NPI and Contracting Eligibility

  Committee Review and Approval

  Add Provider to Network and Trigger Onboarding

  Agents and Roles:

  👔 Senior Project Manager: Define scope, milestones, dependencies, and documentation standards.

  💎 Senior Developer: Architect backend workflow automation (API calls, credentialing logic, audit trails).

  🎨 UI Designer: Create credentialing dashboard, provider portal, and committee review interface.

  🧪 Experiment Tracker: Plan A/B tests for onboarding efficiency and automation accuracy.

  📸 Evidence Collector: Validate data integrity and compliance with CMS/state rules.

  🔍 Reality Checker: Ensure production readiness, scalability, and HIPAA compliance.

  Deliverables:

  System architecture diagram

  Credentialing workflow automation scripts

  UI mockups for provider and admin portals

  Compliance checklist (HIPAA, CMS, state)

  Documentation and deployment plan

  Quality Gates:

  Audit‑ready credentialing records

  Automated CAQH/NPI/license verification

  Committee approval workflow with digital signatures

  Integration with provider onboarding system
